Publish and share your own website for QA with GitHub

Step 1: Setup a GitHub Account
Join Github by clicking the link to create an account. If you already have an account you can skip this step.
https://github.com/join

Step 2: Download the Github Desktop App
Downloading this app will make it a little easier to deploy and manage changes to the code of the website.
Windows:
https://central.github.com/deployments/desktop/desktop/latest/win32
macOS:
https://central.github.com/deployments/desktop/desktop/latest/darwin
