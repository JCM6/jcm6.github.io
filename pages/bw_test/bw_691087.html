Bug 691087: [Branded Web] Checkboxes Are Invisible when Primary Color is White
Reproduced On:jcm6.github.io/pages/bw_test/bw_691087.html
Special Setup Instructions:
1. Login to the Branded Web Dashboard.
2. Open Site Settings.
3. Set the Background Color to: #ffffff.
4. Set the Primary Color to: #ffffff.

Steps to Recreate
1. Click My Account.
2. Create a New Account.
3. Check boxes to determine Notification Settings.
4. Save the Account.
